# Lesson 01 — Sounds + Korean Cheat Codes (Unit 0)

Goal: pronounce Japanese convincingly using sounds you already have, and install
the Korean→Japanese grammar map so every later lesson is just vocabulary.

---

## Part 1 — The 5 sounds Koreans get wrong

Everything else in Japanese maps cleanly onto Korean sounds. Only these need work:

### 1. つ (tsu) — 쓰 아님, ㅊ 아님

Korean has no "ts". It's 츠 said with the tongue touching the teeth like ㅆ.
- つぎ (tsugi, 다음) → "츠기" but the 츠 starts like ㅆ
- Drill word: 「いくつ」 (ikutsu, 몇 개) — 이쿠츠

### 2. ざ・ず・ぜ・ぞ (z sounds) — ㅈ 아님

Korean ㅈ is "j". Japanese ざ is a buzzing "z" like English "zoo".
- ぜんぶ (zenbu, 전부) → "젠부" ❌ → "zㅔㄴ부" ✅ (벌이 윙윙거리는 z)
- Drill word: 「ください」는 아니고... 「ぜんぜん」 (zenzen, 전혀)

### 3. Long vowels — 길이가 뜻을 바꿈

Korean ignores vowel length. Japanese does not:
- おばさん (obasan, 아줌마) vs おばあさん (obaasan, 할머니) — 잘못 부르면 실례
- ビル (biru, 빌딩) vs ビール (biiru, 맥주) — 이건 골프장에서 진짜 중요함
- Rule: 로마자에 모음이 두 개면 (aa, ii, uu, ee, oo/ou) 진짜로 두 박자 끌기

### 4. Small っ — 받침처럼 끊기

きって (kitte) = "킷테". 한국어 받침 ㅅ/ㄱ처럼 한 박자 멈춘다.
- ちょっと (chotto, 좀/잠깐) → "춋토"
- まって (matte, 기다려) → "맛테"

### 5. ん before b/p/m — ㅁ으로 변신

- がんばって (ganbatte) → 실제 발음 "감밧테"
- せんぱい (senpai) → "셈파이"

**Everything else — あかさたな 전부 — 한국어 소리로 충분하다. 억양만 평평하게.**
Japanese is flat compared to Korean. 튀는 강세를 주지 말 것.

## Part 2 — The particle map (이걸로 문법 끝)

한국어 문장을 그대로 두고 조사만 바꾸면 일본어가 된다:

| Korean | Japanese | Sound | Example |
|--------|----------|-------|---------|
| 은/는 | は | 와(wa) | 저는 → わたしは (watashi wa) |
| 이/가 | が | 가(ga) | 맥주가 → ビールが (biiru ga) |
| 을/를 | を | 오(o) | 맥주를 → ビールを (biiru o) |
| 에 (장소/시간) | に | 니(ni) | 3시에 → さんじに (sanji ni) |
| 에서 (행동 장소) | で | 데(de) | 여기서 → ここで (koko de) |
| 도 | も | 모(mo) | 저도 → わたしも (watashi mo) |
| 의 | の | 노(no) | 저의 것 → わたしの (watashi no) |
| 부터 | から | 카라(kara) | 9시부터 → くじから (kuji kara) |
| 까지 | まで | 마데(made) | 역까지 → えきまで (eki made) |
| 와/과/하고 | と | 토(to) | 맥주하고 → ビールと (biiru to) |

Proof it works — build this sentence Korean-style:

- 한국어: 저는 / 맥주를 / 주세요
- 일본어: わたしは / ビールを / ください (watashi wa biiru o kudasai)

같은 순서. 조사만 치환. 이게 전부다.

## Part 3 — 20 instant cognates (이미 아는 단어들)

한자어라서 발음만 살짝 다르다. 오늘부터 바로 쓸 수 있음:

| Korean | Japanese | Sound |
|--------|----------|-------|
| 약속 | やくそく | 야쿠소쿠 (yakusoku) |
| 준비 | じゅんび | 즘비 (junbi) |
| 무리 | むり | 무리 (muri) |
| 요리 | りょうり | 료-리 (ryouri) |
| 도착 | とうちゃく | 토-챠쿠 (touchaku) |
| 출발 | しゅっぱつ | 슛파츠 (shuppatsu) |
| 예약 | よやく | 요야쿠 (yoyaku) |
| 안내 | あんない | 안나이 (annai) |
| 안전 | あんぜん | 안젠 (anzen) |
| 신호 | しんごう | 신고- (shingou) |
| 시간 | じかん | 지칸 (jikan) |
| 온천 | おんせん | 온센 (onsen) |
| 간단 | かんたん | 칸탄 (kantan) |
| 최고 | さいこう | 사이코- (saikou) |
| 유명 | ゆうめい | 유-메- (yuumei) |
| 전부 | ぜんぶ | 젠부 (zenbu) |
| 주의 | ちゅうい | 츄-이 (chuui) |
| 무료 | むりょう | 무료- (muryou) |

패턴 감 잡기: 받침 ㄱ → く/き (약속→야쿠소쿠), 받침 ㄴ → ん (시간→지칸),
받침 ㄹ → つ/ち (출발→슛파츠). 모르는 한자어는 이 규칙으로 찍어도 절반은 통한다.

## Drill (Claude와 함께 — METHOD.md 세션 규칙 적용)

전부 production-first: 한국어 프롬프트 → 소리 내서 일본어로 답 → 그 다음에 정답 확인.
객관식 없음, 보고 고르기 없음.

1. 발음 체크 (shadowing): いくつ, ぜんぜん, ビール vs ビル, ちょっと, がんばって
2. 조사 치환: "저는 한국에서 왔어요" → (직접 생성) → "watashi wa kankoku kara kimashita"
3. 코그네이트 퀴즈: 한국어 단어 제시 → 일본어 발음을 규칙으로 찍기
4. 시나리오 롤플레이: Claude가 편의점 점원 역할 (일본어로) → "스미마셍, 이거 쿠다사이"로 돌파

**Shadowing lines** (따라 말하기, 리듬 평평하게):
- すみません、これください。(스미마셍, 코레 쿠다사이)
- ありがとうございます。(아리가토- 고자이마스)
- だいじょうぶです。(다이죠-부데스)
