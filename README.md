# Japanese Lessons — Henry

## Goal

Conversational survival Japanese for a Korean tourist in Japan. Target situations:

- Buying things (konbini, shops, pro shop)
- Ordering food and drinks
- Golf course: check-in, rentals, caddie, cart, clubhouse, onsen
- Casual small talk with Japanese golfers on the course

**Not** goals: reading kanji, writing, business Japanese, JLPT. Kana appears in lessons only as reference — everything is drilled by sound (hangul pronunciation + romaji).

## Method

You already speak Korean, which is the single biggest cheat code for Japanese:

- Same word order (SOV). You never have to re-think sentence structure.
- Particles map almost 1:1 — 은/는→は(wa), 이/가→が(ga), 을/를→を(o), 에→に(ni), 도→も(mo), 부터→から(kara), 까지→まで(made).
- Huge shared Sino vocabulary: 약속→やくそく(yakusoku), 준비→じゅんび(junbi), 무리→むり(muri), 요리→りょうり(ryouri), 도착→とうちゃく(touchaku).

So lessons skip grammar theory and go straight to: **pattern + swap words + drill out loud**.

## How a session works

1. Open the current lesson in `lessons/`
2. Claude drills you: Korean/English prompt → you answer in Japanese (spoken aloud, typed in romaji or hangul-approximation)
3. Mistakes get logged at the bottom of the lesson file
4. `PROGRESS.md` updated after each session
5. Next session starts with a 2-minute review of previous mistakes

## Files

- `CURRICULUM.md` — the full unit plan and where each unit gets you
- `PROGRESS.md` — what's done, what's shaky, review queue
- `lessons/` — one file per lesson, with drill logs at the bottom
